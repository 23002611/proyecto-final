# Control de Pedidos - Aplicación Electron.js

Este proyecto es una aplicación de escritorio desarrollada con Electron.js que permite llevar el control de empleados, productos y pedidos en un entorno empresarial.

## Características

- **Inicio de Sesión:** Los empleados pueden autenticarse en la aplicación utilizando su ID y contraseña.
- **Gestión de Productos:** Se pueden ver los productos disponibles, editar su información y realizar pedidos.
- **Realizar Pedidos:** Los empleados pueden solicitar productos a proveedores a través de la aplicación.

## Requisitos

- Node.js y npm deben estar instalados en el sistema.
- Se requiere acceso a una base de datos MySQL para almacenar los datos de empleados, productos y pedidos.

## Instalación

1. Clona este repositorio en tu máquina local.
2. Instala las dependencias con el comando `npm install`.
3. Configura la conexión a tu base de datos MySQL en el archivo `main.js`.
4. Ejecuta la aplicación con el comando `npm start`.

## Uso

- **Inicio de Sesión:** Ingresa el ID de empleado y la contraseña para iniciar sesión.
- **Gestión de Productos:** Explora la lista de productos disponibles, edita su información o realiza pedidos.
- **Realizar Pedidos:** Selecciona un producto, elige un proveedor, ingresa la cantidad y confirma el pedido.
